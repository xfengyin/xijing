## 🚀 部署指南

### 本地开发
```bash
cd xijing
npm install
npm run dev
# 打开 http://localhost:5173
```

### 生产构建
```bash
npm run build
# 输出到 dist/ 目录
```

### PWA部署
1. 将 `dist/` 目录部署到支持HTTPS的服务器
2. 确保所有图标文件在 `icons/` 目录
3. 验证 manifest.json 可访问
4. 验证 Service Worker 注册成功

### 推荐平台
- **Vercel** - 自动HTTPS，全球CDN
- **Netlify** - 简单易用，PWA友好
- **GitHub Pages** - 免费托管
