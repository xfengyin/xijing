## ✨ 项目特色

### 声境 · 3D唱片店
- 🎵 Three.js 构建的沉浸式3D唱片店场景
- 💿 可旋转的黑胶唱片机，跟随音乐律动
- ✨ 动态粒子效果与音波可视化
- 🎨 根据歌曲情绪自动变换主题色彩

### 词境 · 歌词可视化 (开发中)
- 📜 逐字歌词同步显示
- 🌟 歌词情感分析与主题适配
- 💬 粉丝留言互动墙

### 忆境 · AI纪念空间 (开发中)
- 🤖 AI音质修复增强
- 🎤 虚拟对唱功能
- 📺 永恒直播间

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/xfengyin/xijing.git
cd xijing

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 🎮 使用指南

### 基本操作
- **播放/暂停**: 点击底部播放器中央的播放按钮
- **切换歌曲**: 点击右侧歌曲列表或左右切换按钮
- **进度控制**: 拖拽进度条调整播放位置
- **键盘快捷键**: 
  - `↑/↓`: 在歌曲列表中导航
  - `Enter`: 播放选中的歌曲

### 3D场景交互
- **鼠标移动**: 视角轻微跟随，营造沉浸感
- **自动动画**: 唱片旋转、粒子浮动、音波扩散

## 🛠️ 技术栈

- **3D引擎**: Three.js
- **构建工具**: Vite
- **动画**: GSAP
- **音频**: Howler.js (预留)
- **样式**: 原生CSS + CSS变量
